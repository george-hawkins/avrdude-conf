REST resources

  • /conf - ids for all programmers and parts.
  • /conf/content - details for all programmers and parts.
  • /conf/programmers/ids - ids for all programmers.
  • /conf/programmers/ids/ - details for programmer with specified id.
  • /conf/parts/ids - ids for all parts.
  • /conf/parts/ids/ - details for part with specified id.
  • /conf/parts/signatures - signatures for all parts.
  • /conf/parts/signatures/ - details for part with specified signature.
